# Calculator_app 
## About
This is a calculator that can perfom basic mathematical operations to solve problems that needs basic calculations with ease. Designed with a clean interface and smooth user experience. 
## Features: 
1. Perform all basic arithmetic operations (Add, Subtract, Multiply, Divide) 2.Responsive and intuitive UI 3.Error handling for invalid inputs 4.Lightweight and fast performance
